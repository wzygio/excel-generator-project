## 3. 行分类标签字典（Col 3 = Factory）

Col 3（Factory / 指标分类标签）承载行级分类，决定了该行各时间列的数据含义：

| 标签值 | 语义 | 数值范围 | 示例 |
|--------|------|----------|------|
| `CT投入量` | CT站别投入数量 | ≥ 0, 整数 | `86225.0` |
| `CT良率` | CT站综合良率 | [0, 1] | `0.9343` |
| `CT_A良率` | CT站 A 品良率 | [0, 1] | `0.8823` |
| `31000_T良率占比` | 31000_T 良率占比 | [0, 1] | `0.0` |
| `CT投入量_MVI不良占比` | CT投入量的 MVI 不良占比 | [0, 1] | `0.206` |
| `CLA2_G品占比` | CLA2 G品占比 | [0, 1] | `0.0` |
| `CLA2投入量` | CLA2投入数量 | ≥ 0, 整数 | `0.0` |
| `CT投入量_AVI不良占比` | CT投入量的 AVI 不良占比 | [0, 1] | - |
| *(更多)* | 其他类似指标 | float | - |

## 4. 数据结构特征

```text
┌───────────────────────────────────────────────────────────────────┐
│  二维交叉表设计                                                    │
│                                                                   │
│  行维度:  (ProductCode × Operation × Factory标签)  约 1529 行     │
│  列维度:  月(M05) → 周(W20) → 逐日(5/4~5/10)  共 9 列时间列      │
│                                                                   │
│  层级下钻: 月度 > 周度 > 逐日 (左→右粒度递增)                     │
│                                                                   │
│  合并单元格:                                                       │
│    Col 1 (ProductCode) 和 Col 2 (Operation) 在同类产品下          │
│    纵向合并，ffill 修复后可用                                     │
│                                                                   │
│  关键修正:                                                        │
│    Col 5 表头显示"项目/日期" → 实际为【不良Code】                 │
└───────────────────────────────────────────────────────────────────┘
```

## 6. 读取代码示例（通过 COM）

```python

excel = win32com.client.Dispatch("Excel.Application")
excel.Visible = False
excel.DisplayAlerts = False

wb = excel.Workbooks.Open(r"<encrypted_file_path>")
ws = wb.Sheets("CT")

# 读取数据
for row in range(1, ws.UsedRange.Rows.Count + 1):
    for col in range(1, ws.UsedRange.Columns.Count + 1):
        val = ws.Cells(row, col).Value
        # process val

wb.Close(SaveChanges=False)
excel.Quit()
```
